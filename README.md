# Reporter-api

A lightweight server to report metrics

Live running server: https://reporter-metrics-api.herokuapp.com

Live running Swagger API documentation: https://reporter-metrics-api.herokuapp.com/api-docs

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn start:dev

# production mode
$ yarn start
```

## Test

```bash
# unit tests
$ yarn test
```
